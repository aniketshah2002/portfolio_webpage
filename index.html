My Projects

AI ANALYTICS JOB PIPELINE DASHBOARD

Built a data pipeline to find out what skills are really in demand for data jobs. The Python script scrapes live job postings, uses spaCy (NLP) to read the descriptions and extract tech keywords, and then loads all the clean data into Google BigQuery. Then built a Looker Studio dashboard to visualize the trends, showing how often AI skills pop up compared to traditional ones. The project culminates in an interactive Looker Studio dashboard that visualizes trends and answers the question: "How integrated is AI in the data analytics industry today?"

AI API Integration (SerpApi) Google Cloud Platform Google BigQuery Looker Studio GitHub ETL Pipeline Development

NIFTY 50 STOCK ANALYSIS DASHBOARD

Engineered a complete, automated ETL pipeline to analyze NIFTY 50 stock trends, starting with API-style data extraction using Python (yfinance, pandas). This data was loaded directly into a Google BigQuery cloud data warehouse, where I used advanced SQL (Window Functions, CTEs) to transform raw prices into complex technical indicators like Bollinger Bands and "Golden Cross" signals. Finally, I built a dynamic, interactive BI dashboard in Looker Studio to visualize these insights and empower data-driven analysis.

Data Engineerng Pipeline Development Google Cloud Platform Data Modelling Looker Studio SQL BigQuery

E-COMMERCE SALES INTERACTIVE DASHBOARD

Designed and built an interactive e-commerce sales dashboard using SQL for data preparation and Tableau for visualization. The project involved transforming raw data from multiple tables by cleaning and joining it with SQL to create a single, analysis-ready dataset. The final dashboard visualizes core business metrics, including sales trends over time, top-selling product categories, and key geographical markets to provide clear, actionable insights.

SQL DB Browser Tableau Data Visualization Public Dashboard
